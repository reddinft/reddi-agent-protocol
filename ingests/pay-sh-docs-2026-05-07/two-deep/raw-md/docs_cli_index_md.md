# CLI Reference

> Grouped reference for the pay command-line interface.

The CLI reference is grouped by workflow instead of one page per command.

## Agent summary

- Use global flags before the subcommand.
- Prefer sandbox examples unless mainnet is explicitly requested.
- Use grouped pages to find the command family, then preserve command-specific arguments.

## Groups

- [Global flags](/docs/cli/global-flags): network, account, output, approval, and debugger flags.
- [HTTP clients](/docs/cli/http-clients): `pay curl`, `pay fetch`, `pay wget`.
- [Agents](/docs/cli/agents): `pay claude`, `pay codex`, `pay mcp`.
- [Provider catalog](/docs/cli/provider-catalog): `pay skills`.
- [Gateways](/docs/cli/gateways): `pay server`.
- [Wallets](/docs/cli/wallets): `pay account`, `setup`, `topup`, `solana`, `send`.
